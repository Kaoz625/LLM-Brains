# Comet Browser — Perplexity AI's Native AI Browser

**Source:** https://www.perplexity.ai/comet, Wikipedia, 9to5Mac, beam.ai
**Date:** 2026-04-09

## Overview
Comet is an AI-powered web browser built on Chromium, developed by Perplexity AI. Released for Windows/macOS (July 2025), Android (Nov 2025), iOS (March 2026).

## Key Features
- Integrated Perplexity answer engine — live web search AI built into the browser
- Hybrid search: simple queries get instant results, complex questions use Perplexity's answer engine with citations
- Deep Research: scans multiple sources simultaneously, provides summarized answers with references
- Agentic capabilities: multi-step task automation (calendar events, LinkedIn lookups, meeting briefs)
- Emphasizes what happens AFTER a page loads, not just rendering speed

## Pricing
- Originally $200/month at launch
- iOS: free with Pro ($20/month) and Max tiers available

## Why It Matters
Comet represents the shift toward "agentic browsers" — browsers that don't just display web pages but actively assist with tasks. Direct competitor to Google's Dia browser and Arc.
